Solve the maze in the maze.png to get the password.

Grids are provided for reference.

Find the binary sequence using following rules:

You enter the maze from top opening of the maze and exit through the bottom.
Each vertical move of one gridblock is taken as 1.
Each horizonal move one gridblock is taken as 0.
Since entering and exiting moves are both vertical, they're considered as 1.

Example:
Solution for sample.png maze is : 1010001011

The binary sequence that you get from solving the maze.png is the password for next level.
